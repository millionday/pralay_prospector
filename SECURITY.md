# Security Policy

## Supported Versions

| Version | Supported |
|---|:---:|
| 2.x | ✅ |
| 1.x | ❌ |

## Reporting a Vulnerability

If you discover a security issue (e.g. XSS in exported data, unsafe eval, unintended data exfiltration), please **do not open a public issue**.

Email: **mishrakeshav554@gmail.com** (or open a private GitHub security advisory)

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 72 hours and credit you in the changelog if you wish.

---

*Pralay Prospector runs entirely client-side. No data is ever sent to external servers by this extension. All scraped data is stored in `chrome.storage.local` on your own machine.*
